# 👩‍⚕️ Landing Page - Flavia Fantim

> Uma landing page personalizada, elegante e funcional desenvolvida para apresentação profissional e conversão de leads.

## 🔗 Demonstração
**Veja o projeto online:** [Acesse aqui](https://landing-page-flavia-fantim.vercel.app/)

---

## 💻 Sobre o Projeto
Este projeto foi focado na criação de uma presença digital sólida para a Flavia Fantim. O objetivo principal foi equilibrar a estética visual com a clareza das informações, garantindo que o usuário encontre facilmente os serviços oferecidos e os canais de contato. A interface foi pensada para transmitir autoridade e confiança.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica para melhor acessibilidade.
- **CSS3:** Estilização moderna com foco em uma paleta de cores harmoniosa.
- **Responsividade:** Layout totalmente adaptado para uma leitura confortável em qualquer dispositivo.
- **Vercel:** Deploy e hospedagem.

## 🎨 Diferenciais Técnicos
- **Design Personalizado:** Interface desenvolvida para refletir a identidade profissional da cliente.
- **UX (User Experience):** Fluxo de navegação simplificado para facilitar o agendamento de consultas ou serviços.
- **Otimização de Carregamento:** Código limpo para garantir que a página abra rapidamente no celular.
